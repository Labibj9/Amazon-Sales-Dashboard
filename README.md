Amazon Sales Analysis Dashboard
Overview

This project is a Power BI dashboard that analyzes Amazon product sales data. It provides insights into sales performance, product trends, and customer engagement to support better decision-making.

Key Features
YTD Sales, QTD Sales, Products Sold, and Reviews
Monthly and weekly sales trends
Top 5 products by sales and reviews
Category-wise sales analysis
Interactive filters (Product Category, Quarter)
Tools Used
Power BI
Excel / CSV (Data Source)
